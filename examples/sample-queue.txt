# ============================================
# New Session Markers
# ============================================

# Start a fresh Claude Code session (clears conversation history)
# Can be used alone (>>>) or with a prompt (>>> prompt)
>>>
Say "Fresh session started!" to confirm.

# Or combine with prompt on same line:
>>> This is another new session. Confirm by saying "New session started!"

# ============================================
# Multiline Prompts
# ============================================

# Multiline prompt using >>( and >>)
>>(
Please create a simple function that:
1. Takes a number as input
2. Returns the square of that number
3. Include a brief comment

Just show the code, no explanation needed.
>>)

# ============================================
# Session Labels
# ============================================

# Save current session with a label (can be restored later)
>>{Label:test-checkpoint}

# Simple prompt after labeling
Confirm the session was labeled by saying "Session labeled!"

# ============================================
# Load Session (uncomment to test)
# ============================================

# Load a previously saved session
# >>{Load:test-checkpoint}

# Load session and start fresh conversation
# >>>{Load:test-checkpoint} Continue from the checkpoint

# ============================================
# Multiline with New Session
# ============================================

# Multiline prompt that also starts a new session
>>>(
This is a multiline prompt in a new session.
Please confirm by listing:
- "New session: yes"
- "Multiline: yes"
>>)

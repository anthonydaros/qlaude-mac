# qlaude Sample Queue File
# Copy this to .qlaude/queue in your project root to test all features
# Lines starting with # are comments (not processed)

# ============================================
# Basic Queue Items
# ============================================

# Simple prompt (bare text = prompt)
Say "Hello from qlaude queue!" and nothing else.

# Another simple prompt
What is 2 + 2? Answer with just the number.

# ============================================
# Pause Points
# ============================================

# @pause pauses auto-execution until you manually resume
# Use :resume or Telegram /resume to continue
@pause Checkpoint: Review the responses above before continuing

# ============================================
# New Session Markers
# ============================================

# Start a fresh Claude Code session (clears conversation history)
# @new is standalone — put the prompt on the next line
@new
Say "Fresh session started!" to confirm.

# ============================================
# Multiline Prompts
# ============================================

# Multiline prompt using @( and @)
@(
Please create a simple function that:
1. Takes a number as input
2. Returns the square of that number
3. Include a brief comment

Just show the code, no explanation needed.
@)

# ============================================
# Session Save/Load
# ============================================

# Save current session with a label (can be restored later)
@save test-checkpoint

# Simple prompt after saving
Confirm the session was saved by saying "Session saved!"

# ============================================
# Load Session (uncomment to test)
# ============================================

# Load a previously saved session
# @load test-checkpoint

# ============================================
# New Session + Multiline (separate directives)
# ============================================

# New session marker followed by a multiline prompt
@new
@(
This is a multiline prompt in a new session.
Please confirm by listing:
- "New session: yes"
- "Multiline: yes"
@)

# ================================= ===========
# Selection Prompt Test
# ============================================

# This prompt will trigger a permission dialog (Y/n selection)
# qlaude detects this as SELECTION_PROMPT state and pauses auto-execution
# You can respond via keyboard or Telegram inline buttons
Write a file called "qlaude-test.txt" with the content "Hello from qlaude!"

@pause After permission prompt - verify file was created or declined

# ============================================
# QUEUE_STOP Test (uncomment to test)
# ============================================

# QUEUE_STOP is triggered when:
# 1. Claude outputs the literal text "QUEUE_STOP" (you can ask Claude to say it)
# 2. Rate limit is detected (429 errors, "rate limit" messages)
# 3. Task failure patterns are detected
#
# When QUEUE_STOP is detected, qlaude pauses and the current item
# is re-queued at the front for retry after :resume
#
# Uncomment below to test (will pause the queue):
# Say the exact text "QUEUE_STOP" and nothing else.

# ============================================
# Model Switching
# ============================================

# @model sends /model command to Claude Code to switch models
# @model <name> switches to specified model
# @model (no args) is silently skipped
@model sonnet
Say "I am running on Sonnet!" and nothing else.

# ============================================
# Delay Between Items
# ============================================

# @delay <ms> pauses queue execution for the specified milliseconds
# Useful for rate limit avoidance or pacing long queues
@delay 3000
Say "Waited 3 seconds before this prompt!" and nothing else.

# ============================================
# Escaping @ prefixed text
# ============================================

# Use \@ to escape prompts that start with @ (so they aren't parsed as directives)
\@username mentioned this — this is a regular prompt, not a directive

# ============================================
# Final Pause Point
# ============================================

@pause End of sample queue - all features tested!
